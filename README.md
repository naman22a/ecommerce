# 🛒 Ecommerce Webiste

![Home](./assets/home.png)

## 🐈 Features

-   Login
-   Register
-   Forgot Password
-   Confirmation Email
-   Add to cart
-   Remove from cart
-   Change Quantity of an item in cart
-   Products page with Pagination
-   Product details page
-   Checkout with stripe
-   Orders History
